# Starter - Sails.js coffee

Sails.js starter, collection for most usage library.

## pre-install

 * node.js
 * npm
 * sass
 * compass
 * bootstrap
 * sails.js@0.11.0rc

## how to start

```
npm i && coffee app.coffee
open http://localhost:1337
```

##include

 * jade
 * coffeescript support
 * sass config
 * include compass
 * include bootstrap (sass)
